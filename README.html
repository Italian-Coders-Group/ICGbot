Whats this?


This a discord bot, with the possibility to be extended directly from discord, by using code blocks or .py files. Those extensions, called modules, are the real deal with this bot, as they can do anything, from implemnting a ban command, to a database-backed chat game!

Ok cool, but how do i implement a command?


Simple! in a code block or .py file, you make a coroutine like this:

@modules.Command
async def commandName(msg: discord.Message):
  pass  # do stuff with the message

pretty simple uh?
the modules file will be imported automatically by the bot, so no worries about it!
